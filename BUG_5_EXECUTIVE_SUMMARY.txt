================================================================================
                      FLUXO DE USO END-TO-END
================================================================================

1. LOGIN
   → Usuario digita credenciais
   → Token JWT armazenado

2. NAVEGAR PARA CLIENTES
   → Click aba "Clientes"
   → CrmTab renderiza com lista vazia ou existente

3. CRIAR CLIENTE
   → Click "+ Novo Cliente"
   → Form de 17 campos aparece
   → Preenche dados
   → Click "Adicionar"
   → POST /crm/create
   → Cliente aparece na lista

4. EDITAR CLIENTE
   → Click no cliente da lista
   → Detalhes expandem
   → Click em qualquer campo
   → EditField ativa (inline input)
   → Altera valor
   → Press Enter (ou click OK)
   → PUT /crm/:id
   → Campo salva e volta ao normal

5. DELETAR CLIENTE
   → Cliente aberto nos detalhes
   → Click "Deletar Cliente" (botão vermelho)
   → Confirmação: "Tem certeza?"
   → Click OK
   → DELETE /crm/:id
   → Cliente desaparece da lista

6. VALIDAR PERSISTÊNCIA
   → Press F5 (reload page)
   → Re-login automático
   → Clientes ainda presentes
   → Todas edições persistiram
