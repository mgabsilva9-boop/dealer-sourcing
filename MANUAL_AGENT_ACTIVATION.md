# 🚀 MANUAL AGENT ACTIVATION COMMANDS

Use os comandos abaixo para ativar os agentes paralelos.

---

## COPIE E COLE CADA COMANDO (em paralelo)

### Command 1: Ativar @qa
```
@qa *code-review dealer-sourcing --mode light --yolo
```

### Command 2: Ativar @dev
```
@dev *improve-code-quality src/ --mode yolo
```

### Command 3: Ativar @architect
```
@architect *validate dealer-sourcing --mode interactive
```

### Command 4: Ativar @devops
```
@devops *ci-cd dealer-sourcing --mode yolo
```

## Ou use o ORCHESTRADOR:

```
@aios-master *workflow dealer-sourcing-mvp YOLO
```

Este comando deve iniciar o workflow paralelo completo se @aios-master conseguir ler o `.aios-core/workflows/dealer-sourcing-mvp.yaml`.
