   Response: 200 OK { total, limit, offset, results: [...] }
   
4. GET /sourcing/:id
   Response: 200 OK { id, make, model, year, price, ... }
   Response: 404 Not Found if vehicle doesn't exist
   
5. POST /sourcing/:id/interested
   Body: { notes?: "string" }
   Response: 201 Created { id, vehicle_id, status, saved_at }
   Response: 409 Conflict if already interested
   
6. GET /sourcing/favorites?status=interested&limit=20&offset=0
   Header: Authorization: Bearer <JWT_TOKEN>
   Response: 200 OK { total, limit, offset, results: [...] }

================================================================================
TROUBLESHOOTING
================================================================================

DEPLOYMENT FAILS:
  1. Check build logs for errors
  2. Verify npm install works locally
  3. Check all environment variables are set
  4. Ensure package.json has "start" or correct startCommand

SERVICE CRASHES:
  1. Check DATABASE_URL is set correctly
  2. Verify PostgreSQL service is running
  3. Check JWT_SECRET is set
  4. Review logs: Dashboard → Logs

SLOW RESPONSES:
  1. Starter PostgreSQL may be slow
  2. Check for connection pool exhaustion in logs
  3. Upgrade plan if needed

DATABASE CONNECTION ERROR:
  1. Verify PostgreSQL status is "Running"
  2. Check DATABASE_URL contains correct host/port
  3. Ensure sslmode=require in connection string
